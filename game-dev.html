Game Development Journey

So I've decided that this page will be where I sort of update and keep track of my game development journey. I will be making updates on the game dev stuff I'm learning and any plans I have for a game I want to make. I actually started on Godot originally, but I switched to Unity because I want to switch to Unreal Engine in the future and apparently the coding languages are slightly similar and Unity is still pretty popular and has good documentation and tutorials.

I do actually have a game idea in mind that I want to make, but I want to learn the basics first obviously. It's heavily inspired by "Darkwood", a game I love. The atmosphere and gameplay is so cool despite being a top down 2d game. For this I'll be using Unity, which is not only great for 2D but Darkwood itself was made on Unity. It'll also be a top down game using the same lighting mechanics as Darkwood. I'll be using a lot of the same ideas but a different setting, story and some different mechanics.
